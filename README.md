## Compilacion
Ubicado en la carpeta raiz del proyecto correr el comando por consola:
````sh
$> make all
````
Se generará el ejecutable _stegobmp_

## stegobmp
Modo de uso:
````sh
$> ./stegobmp [OPTIONS]
````
Para ver un menu de ayuda del programa

````sh
$> ./stegobmp -h
````
